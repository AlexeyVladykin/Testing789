# 🚀 Конструктор ракеты

> Собирай. Разбирай. Меняй местами. Никакой сложности — только чистое удовольствие от сборки.

---

## Что это?

Простой визуальный конструктор, где ты собираешь ракету из деталей. Без заумных механик, таймеров и прочей ерунды. Открыл → выбрал деталь → поставил → готово.

Всего на **Python + tkinter**. Никаких внешних зависимостей.

---

## Как запустить

```bash
python rocket_builder.py
```

Всё. Если есть Python 3 — заработает.

---

## Как играть

1. **Справа — каталог деталей.** Носы, корпуса, двигатели, стабилизаторы. По 5 вариантов в каждой категории.
2. **Кликаешь на деталь** — она выбирается.
3. **Кликаешь на сегмент ракеты** — деталь устанавливается.
4. **Красная обводка** — деталь не на своём месте (например, двигатель в слоте для носа).
5. **Зелёная обводка** — всё правильно.

Ещё можно:
- **«Снять деталь»** — убирает деталь из выбранного слота
- **«Сбросить ракету»** — начинает сборку заново
- **«Проверить сборку»** — покажет, насколько всё правильно

---

## Особенности

- **Визуальная модель** — каждый тип деталей рисуется по-своему. Острый нос выглядит не как тупой, большой двигатель отличается от маленького.
- **Свободная сборка** — можно поставить любую деталь в любой слот. Хочешь двигатель сверху? Пожалуйста. Только не удивляйся красной рамке.
- **Компактный интерфейс** — ничего лишнего, всё под рукой.
- **Скролл в каталоге** — даже если деталей много, всё поместится.
